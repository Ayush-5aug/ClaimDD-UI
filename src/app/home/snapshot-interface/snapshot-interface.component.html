SNAPSHOT - INTERFACE

ClaimDD is a very user-friendly Web Application that helps the user evaluate the Delay and/or Disruption cost by uploading the necessary data as an Excel file. Thus, a user does not require to enter a vast amount of data into the Web Application.


CLAIMANT AND PROJECT

The Users can create any number of Claimants and Projects.

One user can manage any number of Projects for the same Claimant.

Note: Number of projects is subject to the license limit.

FAQ: Who can use it? Answer: It can be used by the Contractor, Subcontractor, Claims Consultant, Consultant, and Client.

SAVE AND EXPORT THE CLAIM

The Users can save the evaluated Cost Claim and Claim Report in the cloud (Web Application). Thus, the User can view such a report at a later stage.

Moreover, the user can export Cost Claim as an Excel and Claim Report as a Word file.

FAQ:Can the User share the saved Cost Evaluation and Cost Claim Report with a third party?Answer: No, the User cannot share the saved Cost Evaluation and Cost Claim Report as it is stored in ClaimDD. However, the User can create an account for such a third party and grant access in ClaimDD as a User (Reviewer). The User (Reviewer) can only view such details but cannot modify the original details. Further, such a third party can not view the Project's other data.

CURRENCY

The Users can fix any desired currency for the Project.

Different currency can be set for different projects. Thus, the same user can manage local and international projects.

FAQ: Do same user can fix different currency for national and international project? . Answer: Yes, same user can fix different currency for the local and international project.

FORMAT

The Users can set the date and number format. Further, the limit of decimal places of the numbers can also be set.

Each Project can have its own format settings. Thus, the same user can manage local and international projects..

FAQ: Can the same user set different formats for a national and international project? Answer: Yes, same user can set different format for local and international project.

LICENSE

ClaimDD's license is provided based on the Number of Projects and Projects Value Limit.

One ClaimDD's license has four types of Users such as:

1.Owner

2.Manager

3.Executer, and

4.Reviewer
