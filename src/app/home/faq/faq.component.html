FREQUENTLY ASKED QUESTIONS




You may have some questions for that you may find the answers.


Good Faith

1. Who is Good Faith?

Good Faith is a limited liability company. Good Faith departments are Contract Audit Bureau and Contract Software Bureau.

Good Faith – Contract Audit Bureau - primary service is Contract Auditing Services. Moreover, it provides Contract Management Services that include claims and dispute management. Further, it provides a Contract Counsel Programme (Corporate Training) to empower the contract knowledge.

Good Faith – Contract Software Bureau – is a department of Good Faith. It researches and develops web applications (web-based software) for simplifying contract expert tasks. It has launched a Web Application 1. ClaimDD.

2. Does Good Faith provide Claims Consultant Services?

Yes, Good Faith provides Claims Consultant Services in addition to its primary service, Contract Auditing. Further details can be found at www.GoodFaith.LLC.

3. Does Good Faith develop custom contract software for the user?

Good Faith can develop custom software to carry out contract tasks effectively as required by the user. However, it shall develop such custom software only if it would provide a global solution.

4. Can Good Faith help to justify the Delay and Disruption Cost evaluated using ClaimDD?

Good Faith’s Contract Audit Bureau can help justify such a cost claim. However, as such service is not part of the subscription user shall pay applicable fees to Good Faith.

ClaimDD

1. What is the purpose of ClaimDD?

ClaimDD is a Web Application (web-based software) that helps the user evaluate the Delay and/or Disruption Cost incurred by the Claimant for any event.

2. Who Can Use ClaimDD?

ClaimDD is data-driven web application. Thus, it can be used by anyone who wants to evaluate the Delay and/or Disruption Cost incurred by the Claimant. It can be used by the Contractor, Subcontractor, Claims Consultant, Consultant, and Client.

3. Can I have a Demo for ClaimDD?

Yes, demo for ClaimDD can be arranged upon the request. The request can be sent via Demo Request.

4. What is the limit of the number of Projects?

ClaimDD can be used for any number of Projects subject to purchased Project Count.

5. How much does ClaimDD cost?

ClaimDD cost is depends on number of Projects (Project Count) and maximum value of the Project (Project Value Limit). It is available monthly, quarterly, half-yearly, and yearly subscriptions.

The cost can be evaluated under the Pricing page .

6. Is there any free plan?

Yes, ClaimDD is available for free for 3 months under trial license. The trial license can be used for one (1) project only.

7. What is the difference between various subscription periods such as monthly, quarterly, half-yearly and yearly?

You can choose to pay the subscription for the period of one month, 3 months, 6 months or annual (12 months) at a time. The monthly (unit) price differs for each subscription period (where you can save up to 20%). Please be noted that you can import the data only the subscription period.

8. What happens when I reach the Project Count?

When you used all the purchased Project Count, you shall no longer be able to add a new Project. You either need to upgrade the plan or deactivate the existing Project's license key to add a new Project.
